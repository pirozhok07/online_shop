Backend Developer | Проект: REST API Online Store (FastAPI)

    Разработал асинхронное API на FastAPI с использованием SQLAlchemy 2.0 и асинхронного драйвера aiosqlite, обеспечив высокую пропускную способность запросов.
    Реализовал систему безопасности: JWT-аутентификация и кастомное хеширование паролей с «солью» (Salted Hashing) для защиты данных пользователей.
    Спроектировал структуру БД с использованием связей One-to-Many, реализовав логику корзины покупок и динамическую фильтрацию товаров по категориям и цене.
    Внедрил систему миграций Alembic, что позволило версионировать схему базы данных и безопасно обновлять структуру таблиц.
    Покрыл проект интеграционными тестами с использованием Pytest и HTTPX, реализовав изоляцию тестов через подмену зависимостей (Dependency Overrides) и использование In-memory SQLite.
    Контейнеризировал приложение с помощью Docker и Docker Compose, обеспечив быструю развертку проекта в любой среде.
